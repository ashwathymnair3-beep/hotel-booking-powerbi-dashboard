# Hotel Booking Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard for analyzing hotel booking data. The dashboard provides insights into bookings, revenue, customer behavior, and hotel performance.

## Tools Used
- Power BI
- DAX
- Excel Dataset

## Dataset
Hotel booking dataset containing information about:
- Booking dates
- Hotel type
- Customer details
- Revenue
- Booking status

Dataset File:
hotel_bookings_updated_2024.xlsx

## Key Insights & KPIs
The dashboard analyzes:

• Total Bookings  
• Total Revenue  
• Average Daily Rate  
• Booking Trends Over Time  
• Hotel Type Comparison  
• Customer Distribution  

## Dashboard Features
Interactive filters allow analysis by:
- Hotel type
- Booking date
- Customer category

## Project Files
project_hotel.pbix → Power BI dashboard file  
hotel_bookings_updated_2024.xlsx → Dataset used for analysis

## How to Use
1 Download the `.pbix` file  
2 Open in **Power BI Desktop**  
3 Explore the interactive dashboard
